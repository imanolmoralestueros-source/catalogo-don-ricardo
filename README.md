# Catálogo de exportación - Agrícola Don Ricardo

Proyecto del segundo módulo del curso Fullstack de Tecsup.
Es un catálogo web de productos de exportación de la empresa
(uvas de mesa, arándanos y cítricos), hecho con HTML, CSS y JavaScript.

## Cómo verlo
El proyecto usa módulos de JavaScript, así que hay que abrirlo con un
servidor (Live Server en VS Code, o `python -m http.server`), no con doble clic.
